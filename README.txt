Supported Browsers:
---------------------------
    1. Chrome (Not Tested On Other Browsers)

Run Instruction
---------------------------
    1. After the zip is extracted, right click on 'index.html' file
    2. Then choose "Open With" --> "Chrome"

Required Files
---------------------------
Ensure that the follows files are present in the current directory before opening 'index.html' file.
    1. index.html 
    2. main.js
    3. shared-styles.css
    4. styles.css
    5. variables.css
